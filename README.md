# MetaUniverse Blog

> 与邵承源一起，在电路与系统的边界，探索大模型的星辰大海。

**MetaUniverse** 是一个基于 [VitePress](https://vitepress.dev/) 构建的现代化 AI 技术博客。它不仅是一个静态文档站点，更是一个集成了多种动态交互特性的知识管理系统。

---

## 🚀 Quick Start

```bash
# 环境要求
Node.js 18+
NPM / PNPM

# 安装
npm install

# 运行
npm run dev
```

## ✨ Features

- **🚀 深链接持久化**: 用户设置自动固化。
- **🌗 双重布局模式**: 极简专注模式与标准模式一键切换。
- **🔍 RAG 搜索**: 基于内容的语义检索能力。
- **📝 段落级 Git 注释**: Alt+Click 开启文档深度回溯。
- **🔥 语义热力图**: 视觉化呈现阅读重点。
- **🕸️ 双向知识图谱**: 穿行于 `[[WikiLinks]]` 构建的知识宇宙。
- **📊 张量可视化**: 交互式观测 Transformer 矩阵变换。
- **⚡ WASM 沙箱**: 浏览器端零延迟运行原生 Python。
