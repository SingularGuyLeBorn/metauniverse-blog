# MetaUniverse 博客操作指南

> 探索 AI 的无限可能，构建你的第二大脑。

本指南旨在帮助您快速熟悉 **MetaUniverse** 博客系统的核心功能与操作流程。我们的设计理念是将静态文档与动态交互深度融合，为您提供极致的技术阅读与创作体验。

## 🌟 核心特性概览

MetaUniverse 集成了八大核心技术特性，涵盖了从布局管理到张量可视化的全方位功能。

### 1. 🚀 深链接持久化 (Deep Link Persistence)
您的所有个性化设置（如选定的主题、布局模式等）都会自动保存。即使刷新页面，系统也会恢复到您上次离开时的状态。

### 2. 🌗 双重布局模式 (Dual Layout Mode)
- **默认布局**：左侧导航、中间内容、右侧大纲。
- **专注模式 (Focus Mode)**：一键隐藏所有干扰元素，沉浸于文字。
- **操作**：点击页面右侧悬浮的 **Mode Switcher** 进行切换。该按钮支持自由拖拽。

### 3. 🔍 RAG 语义搜索 (RAG Search)
集成智能搜索能力，支持根据内容相似度进行检索，而不仅仅是关键词匹配。助您精准定位知识。

### 4. 📝 段落级 Git 注释 (Git Annotations)
- **操作**：在阅读过程中，按下 `Alt + 点击` 文本段落。
- **功能**：查看该段落的 Git 修改记录及注释，实现类似代码 Review 的文档协作体验。

### 5. 🔥 语义热力图 (Semantic Heatmap)
页面左侧边缘设有阅读热力条。
- **颜色说明**：蓝色表示快速扫过，红色表示深度停留。
- **价值**：根据您的阅读数据，直观展示文章的重点内容，辅助二次复习。

### 6. 🕸️ 双向知识图谱 (Bidirectional Graph)
基于 `[[WikiLinks]]` 构建。
- **提到 (Mentioned in)**：哪些文章引用了当前页。
- **引用 (References)**：当前页引用了哪些文章。
- **图谱**：文章底部自动生成交互式关系网，快速穿梭于知识宇宙。

### 7. 📊 交互式张量可视化 (Tensor Playground)
内置交互式组件，可实时观测 Transformer 内部的矩阵运算、Attention 机制。点击文章中的 Tensor 卡片即可进入 Playground。

### 8. ⚡ 实时 WASM 沙箱 (Python In Browser)
内置 Pyodide，支持直接在文档中运行 Python 代码。
- **零搭建**：无需后端服务器，代码在浏览器端本地执行。

## ✍️ 创作与编辑

### 实时编辑器 (Live Editor)
在阅读模式下，您可以随时进入 **编辑模式** 修改文章内容。
1. 点击状态栏中的 `👁️ 阅读模式` 切换为 `📝 编辑模式`。
2. 直接在页面上修改文字。
3. 系统将自动同步 (Auto-Sync) 到本地 backend，并触发 Git 自动化提交。

### 公式编辑 (Formula Editing) [新特性]
现在支持更直观的公式源码编辑：
- **操作**：在 `编辑模式` 下，直接点击渲染后的 KaTeX 公式。
- **效果**：公式将展开为原始 LaTeX 源码（如 `$E=mc^2$`），方便您直接修改。
- **渲染**：修改完成后，点击空白处或保存，系统将自动重新渲染。
