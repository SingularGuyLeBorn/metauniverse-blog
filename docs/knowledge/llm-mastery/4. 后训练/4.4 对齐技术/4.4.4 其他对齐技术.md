### CUDA Kernel 伪代码

```cpp
__global__ void 4.4.4_其他对齐技术_kernel(float* x, float* out, int n) {
    int idx = blockIdx.x * blockDim.x + threadIdx.x;
    if (idx < n) {
        out[idx] = __expf(x[idx]);
    }
}
```
